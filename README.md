# MODERATION-BOT-ADMIN-COMMANDS

بوت ديسكورد إداري شامل (عربي + إنجليزي) مع:
- أوامر إدارية كاملة
- نظام صلاحيات 3 درجات بالرتب
- اختصارات عربية وإنجليزية
- الأوامر تعمل مع `!` أو بدون بادئة
- حماية تلقائية ضد السبام/الروابط والـ mentions الخطيرة
- نظام تحقق 6 أرقام خلال 20 ثانية

## الصلاحيات حسب الرتب
### Basic (الرتب الأولى)
- `1480390867205230624`
- `1481126330962546709`

الأوامر:
- `timeout` / `تايم`
- `untimeout` / `انتايم`
- `warn` / `تحذير`
- `unwarn` / `ازالة-تحذير`

### Advanced
- `1481128292126818314`

الأوامر (كل Basic +):
- `kick` / `طرد`
- `purge` / `مسح`
- `slowmode` / `سلومود`
- `lock` / `قفل`
- `unlock` / `فتح`

### Full (أعلى رتبة)
- `1481128292126818314`

الأوامر (كل شيء + أوامر إضافية):
- `ban` / `باند`
- `giverole` / `اعطاء-رول`
- `removerole` / `شيل-رول`
- `say` / `قول`
- `embed` / `امبد`
- `announce` / `اعلان`
- `anonmsg` / `رسالة-سرية` (إرسال رسالة خاصة لعضو بدون كشف اسم المرسل)

> ملاحظة: بحسب الأرقام المرسلة، advanced و full نفس الرتبة حالياً.

## نظام الحماية
- **Anti-Spam**: إذا العضو أرسل عدد رسائل كبير بسرعة، البوت يعطيه تايم تلقائي.
- **Anti-Link/Mass Mention**: يحذف رسائل الروابط و `@everyone/@here` تلقائياً.

## نظام التحقق (6 أرقام خلال 20 ثانية)
عند دخول عضو جديد:
1. البوت يرسل له كود من 6 أرقام بالخاص.
2. يكتب داخل السيرفر: `verify 123456` أو `تحقق 123456` خلال 20 ثانية.
3. إذا صح، يتم فك التقييد عنه ويفتح له الوصول للرومات العامة.
4. إذا انتهى الوقت، ينطرد تلقائياً.

### مهم جداً لإعداد التحقق بشكل صحيح
أنشئ **Restricted Role** في السيرفر (صلاحيات مقفلة للرومات العامة)، ثم ضع الـ ID داخل:
- `VERIFICATION_ROLE_ID` في ملف `.env`

## التشغيل
```bash
npm install
cp .env.example .env
# ضع التوكن + VERIFICATION_ROLE_ID
npm start
```
