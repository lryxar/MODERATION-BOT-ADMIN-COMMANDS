# MODERATION-BOT-ADMIN-COMMANDS

بوت إدارة ديسكورد احترافي بأوامر **Slash Commands** (تبدأ بـ `/`) مع دعم عربي كامل ونظام حماية + تحقق قوي.

## ✅ أهم تغيير
كل الأوامر الآن تبدأ بـ `/` مثل:
- `/timeout`
- `/warn`
- `/announce`
- `/verify`

---

## إضافات قوية (Legendary)
- **نظام تحقق 6 أرقام خلال 20 ثانية**:
  - العضو الجديد يستلم كود بالخاص.
  - يدخل `/verify` ويكتب الكود.
  - إذا تأخر: يطرد تلقائياً.

- **Anti-Link & Anti-Mass-Mention**: حذف تلقائي للروابط و `@everyone/@here`.
- **سجل إداري (Mod Logs)** اختياري داخل قناة مخصصة.
- **فحوصات أمان إضافية**:
  - منع تنفيذ الأوامر على المالك.
  - منع استهداف نفسك.
  - فحص ترتيب الرتب قبل العقوبات.
  - فحص رتبة البوت قبل إعطاء/نزع الرتب.

## إعداد مهم قبل التشغيل
1. فعل **Message Content Intent** و **Server Members Intent** من Discord Developer Portal.
2. أنشئ رتبة مقيدة (Restricted Role) تمنع كتابة/رؤية الرومات العامة حسب نظامك.
3. ضع ID الرتبة في `VERIFICATION_ROLE_ID`.
4. (اختياري) ضع قناة لوجات في `LOG_CHANNEL_ID`.
5. (اختياري) ضع `GUILD_ID` لتسجيل أوامر `/` بسرعة داخل سيرفرك.
